# Public Demo Notes

Use this note when preparing a public demo of the repository.

## Demo Goals

- Show the parent-facing value clearly.
- Explain the automation flow.
- Keep payment and request-limit behavior understandable.
- Use demo values only.
- Keep privacy notes visible.

## Before Sharing

- Check that the README is current.
- Check that screenshots are safe.
- Check that examples do not reveal private family data.
- Check that product notes match the current scope.
